# Hell Maiden Wiki

Independent English guide hub for **Hell Maiden** at
`https://www.hell-maiden.wiki`.

The launch site covers:

- first-run movement, aiming, dash timing and card decisions;
- Spirit Cards, Weapon Cards, Mod Cards, merging and positioning;
- patch-safe build planning and the Queen Minos fight;
- Dante, Virgil, Beatrice, Minos and the Poets of Limbo;
- demo versus Early Access scope and non-transferable progress;
- release date, Windows and console status, Steam Deck compatibility;
- official Steam downloads, system requirements and soundtrack links;
- Article, BreadcrumbList, ItemList, WebSite, Organization, FAQPage,
  VideoGame and VideoObject schema.

## Development

```bash
pnpm install

pnpm next:build
pnpm build
```

## Source policy

- Current Steam facts, patch notes and live tooltips outrank old demo values.
- The official wiki is used for entity and system references without copying its
  prose or presenting demo documentation as permanent Early Access truth.
- YouTube supports visual cross-checks; articles are original decision guides,
  not transcript rewrites.
- No fake codes, unsupported tier lists, cracks, trainers, modified clients,
  APK mirrors or unverified platform promises.

Hell Maiden and official materials belong to their respective rights holders.
